GetDetails<T>

Present under Capsicum namespace. Get detials from an api in a given type

List<T> allTyped=new ConsumeJsonAPI().GetDetails(api);

  • api:- Type of String class. Give here an api url.
  • allTyped:- Type of List<T> this will be return type.
