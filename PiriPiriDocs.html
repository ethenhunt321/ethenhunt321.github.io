Installation

For Visual Studio IDE, the laziest way is to look on nuget package manager. Search with keyword PiriPiri.

Integration

PiriPiri must be integrated in a class file where you want a page object model design pattern.

Coded UI

Implementation must be done in Class with [FindsBy] attribute.

Selenium

Not applicable. [FindsBy] attribute is only applicable for Coded UI.

Specfow

Not applicable. [FindsBy] attribute is only applicable for Coded UI.

NUnit

Not applicable. [FindsBy] attribute is only applicable for Coded UI.

FindsBy

Present under PiriPiri namespace. Decorate Property to find HtmlControls.

[FindsBy(How:"Id", Using:"pass")]

  • How:- Type of string. Give here SearchProperties to find controls like Id (exactally given by coded ui).
  • Using:- Type of string. Give here value of SearchProperties to find controls.

POM

Present under PiriPiri namespace. Finds decorated HtmlControl property from web.

POM pom= new POM(BrowserWindow,Type);

  • BrowserWindow:- Type of BrowserWindow. Give here current instance of browser window.
  • Type:- Type of typeof(Current Class). Give here current class name.
